# Калькулятор на чистом JavaScript

Веб-калькулятор с классическим интерфейсом в стиле iOS, реализованный без каких-либо фреймворков и зависимостей. Поддерживает все основные арифметические операции, историю вычислений, обработку ошибок и управление с клавиатуры.

## Стек

| Слой | Технология |
|------|-----------|
| Frontend | HTML5, CSS3, JavaScript (ES6+) |
| Backend | N/A |
| Database | N/A |

## Функционал

- Арифметические операции: `+`, `−`, `×`, `÷`
- Цепочки вычислений (например, `2 + 3 × 4`)
- Процент (`%`), смена знака (`+/−`)
- Очистка (`C`) и удаление последнего символа (`⌫ Backspace`)
- Обработка деления на ноль
- История вычислений (раскрывается по кнопке)
- Управление с клавиатуры

## Запуск локально

1. Скачайте или клонируйте репозиторий:
   ```bash
   git clone https://github.com/YOUR_GITHUB_USERNAME/YOUR_REPO_NAME.git
   ```
2. Откройте файл `index.html` в любом современном браузере.

> Серверная часть не требуется — проект полностью клиентский.
